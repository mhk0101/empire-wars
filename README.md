# 👑 Empire Wars — راهنمای اجرا روی ویندوز

این یک بازی استراتژیک تلگرامی است که با **Next.js + PostgreSQL** ساخته شده.
برای اجرای آن روی کامپیوتر خودت این مراحل را به ترتیب انجام بده.

---

## ۱) نصب پیش‌نیازها

### الف) نصب Node.js
از سایت زیر نسخه **LTS** را دانلود و نصب کن:
👉 https://nodejs.org

برای اطمینان از نصب، در CMD این را بزن:
```
node -v
npm -v
```
باید شماره نسخه نمایش داده شود.

### ب) نصب PostgreSQL (دیتابیس)
از سایت زیر دانلود و نصب کن:
👉 https://www.postgresql.org/download/windows/

هنگام نصب از تو **رمز عبور (password)** برای کاربر `postgres` می‌پرسد.
**این رمز را یادداشت کن.** (پیشنهاد ساده: `postgres`)

پورت پیش‌فرض را روی `5432` بگذار.

---

## ۲) ساخت دیتابیس

بعد از نصب PostgreSQL، برنامه **pgAdmin** (که همراه آن نصب می‌شود) را باز کن
یا در CMD این دستور را بزن (رمز را وارد کن):

```
psql -U postgres
```

سپس داخل psql این را بزن تا دیتابیس ساخته شود:
```
CREATE DATABASE app_db;
\q
```

> اگر دستور `psql` شناخته نشد، یعنی مسیر PostgreSQL در PATH ویندوز نیست.
> می‌توانی به‌جای آن از pgAdmin → کلیک راست روی Databases → Create → Database استفاده کنی
> و نام `app_db` را بدهی.

---

## ۳) ساخت فایل `.env`

در همان پوشه پروژه (کنار `package.json`) یک فایل به اسم **`.env`** بساز
و این خط را داخلش بنویس (به‌جای `postgres` دومی، رمز خودت را بگذار اگر فرق دارد):

```
DATABASE_URL=postgresql://postgres:postgres@127.0.0.1:5432/app_db
```

> ساختار: `postgresql://کاربر:رمز@127.0.0.1:5432/نام_دیتابیس`

یک فایل نمونه به اسم `.env.example` در پروژه هست؛ می‌توانی از روی آن کپی بگیری.

---

## ۴) نصب کتابخانه‌ها

در CMD وارد پوشه پروژه شو و این دستور را بزن (یک‌بار کافی است):

```
cd C:\Users\MB\Desktop\gameemprator
npm install
```

چند دقیقه طول می‌کشد تا پوشه `node_modules` ساخته شود.

---

## ۵) ساخت جدول‌های دیتابیس

این دستور جدول‌های بازی را داخل دیتابیس می‌سازد:

```
npx drizzle-kit push
```

اگر سوالی پرسید، گزینه‌ای که جدول را می‌سازد (Yes / create) را بزن.

---

## ۶) اجرای بازی

### حالت توسعه (برای تست و بازی):
```
npm run dev
```
سپس در مرورگر برو به:
👉 http://localhost:3000

صفحه اصلی باز می‌شود. روی **«🎮 ورود به بازی»** بزن تا وارد بازی شوی.

### حالت نهایی (Production):
```
npm run build
npm run start
```

---

## ❓ مشکلات رایج

- **`DATABASE_URL is required`** → فایل `.env` ساخته نشده یا خط آن اشتباه است.
- **خطای اتصال به دیتابیس / ECONNREFUSED** → سرویس PostgreSQL روشن نیست
  (در ویندوز: Services → postgresql → Start) یا رمز/پورت در `.env` اشتباه است.
- **صفحه روی «در حال بارگذاری امپراتوری» می‌ماند** → یعنی جدول‌ها ساخته نشده‌اند؛
  دستور `npx drizzle-kit push` را دوباره اجرا کن.
- **`npm` شناخته نشد** → Node.js درست نصب نشده؛ CMD را ببند و دوباره باز کن.
